University Project


TI QUIZ

This is a game we've made to university's final project.

TI Quiz is a multiplayer educational game in real time, developed to be a fun and relaxed study alternative. When we developed the game, the websocket was still a unexplored technology.
In TI Quiz, players access the game's website via computer and mobile device, being the last one used as a joystick. On the computer, the access code is generated, which must be inserted in the mobile devices to enter the match.

Main Contributions:

- System architecture design;
- Gameplay programming on both client and server;
- UI programming;
- Assets creation;
- Game Design Document creation.

Tech Stack:

- Client: HTML5, Canvas
- Backend: Javascript, NodeJS, Websocket
